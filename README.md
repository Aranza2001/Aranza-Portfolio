
# Portfolio Personal - Aranza Díaz

## Descripción General

Este proyecto consiste en el diseño y desarrollo de un **sitio web de marca personal estático**. El objetivo principal es presentar mi identidad digital profesional, consolidando mi portafolio de proyectos académicos y experiencia en el sector tecnológico.

Este sitio cumple con los requerimientos de la asignatura de **Computación Móvil**, implementando HTML5 semántico, CSS3 moderno y diseño responsivo, potenciado por la arquitectura de componentes de **Astro**.

### Propósito y Visión

Más allá de ser un requisito académico, este portfolio ha sido creado como una herramienta real para mi inserción laboral. Actúa como mi carta de presentación digital en mi búsqueda actual de práctica profesional, centralizando mis logros y demostrando mis capacidades técnicas en un formato web accesible.

### Enfoque de Marca Personal

> **"Conectando soluciones técnicas con valor de negocio."**

 Mi perfil está especializado en la intersección entre ingeniería y negocio:

*  **Cloud Solutions Architecture**
*  **Technical Sales & Pre-Sales Engineering**
*  **Desarrollo de soluciones escalables con IA**

##  Tecnologías Utilizadas

El proyecto combina las bases fundamentales del desarrollo web con herramientas modernas para optimizar el rendimiento y la mantenibilidad:

| Tecnología | Uso en el Proyecto |
| :--- | :--- |
| **HTML5** | Estructura semántica (uso de `<header>`, `<main>`, `<article>`, etc.). |
| **CSS3** | Estilos personalizados, Flexbox/Grid y diseño responsivo. |
| **Astro** | Framework principal para la arquitectura de componentes y optimización estática. |
| **Tailwind CSS** | Framework de utilidades para agilizar el diseño y mantener la consistencia visual. |
| **JavaScript** | Lógica para interactividad del cliente (ej. manejo del Tema Oscuro). |

## Instrucciones de Instalación y Despliegue

Este proyecto utiliza **Astro**, por lo que requiere Node.js para gestionar las dependencias y compilar el sitio correctamente.

### 1. Clonar el repositorio
Descarga el proyecto en tu máquina local:
```bash
git clone https://github.com/Aranza2001/Aranza-Portfolio.git
````

### 2\. Instalar dependencias

Instala las librerías necesarias ejecutando:

npm install

### 3\. Ejecutar en modo desarrollo

Para visualizar el sitio en navegador:

npm run dev

Abrir en el navegador: `http://localhost:4321`
